hi, thanks for checking out my City of Boston bus tracker

This project demonstrates using the MapBox GL API, an asynchronous data fetch from the MBTA API to get bus information in json format, creating and manipulating geojson files, plotting the busses on the map, and updating the bus locations every 5 seconds. Hovering the mouse over a bus shows the current occupancy level.
